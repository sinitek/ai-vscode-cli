# Runbooks 目录说明

这里用于沉淀运行、发布、排障、值班和环境操作手册。

## 当前重点文件

- `.ch/docs/runbooks/local-development.md`：本地开发、调试、构建、打包手册。
- `.ch/docs/runbooks/PITFALLS.md`：踩坑后的复盘、复发条件、规避方式与验证方法。

## 未来建议收录

- 测试环境初始化手册
- 发布与回滚手册
- 常见故障排查手册
- 数据修复与审计操作手册

## 维护规则

- runbook 要面向“接手的人或代理立刻能执行”来写。
- 一旦发现真实踩坑，默认要补充到 `PITFALLS.md` 或相应专题 runbook。
- 不要把关键操作只留在聊天或平台 wiki。
